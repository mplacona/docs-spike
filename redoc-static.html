Create a contract execution transaction for developer

Create a contract execution transaction for developer

Authorizations:
bearerAuth
Request Body schema: application/json

Create transaction for developer request

abiFunctionSignature
string

AbiFunctionSignature is the contract abi function signature. e.g. burn(uint256)

abiParameters
Array of strings

AbiParameters is the contract interaction parameters when performing contract interaction. e.g. ["100"].

autoAccelerate
boolean
Default: false

AutoAccelerate enable the Auto acceleration mechanism to raise fee and resend transaction, Default: false.

contractAddress
required
string

ContractAddress is the contract address for execution.

feeLevel
string
Enum: "LOW" "MEDIUM" "HIGH"

FeeLevel is the fee level. LOW / MEDIUM / HIGH - Defines the blockchain fee level which will be paid for the transaction. Required, if gasLimit, maxFee and priorityFee are not provided. See estimate transaction fee API.

gasLimit
string

GasLimit is for EVM chains, the limit for how much can be used. This is obtained from estimate transaction fee API.

gasPrice
string

GasPrice is for EIP-1559 non-support EVM chains only, e.g. Harmony. This is obtained from estimate transaction fee API.

idempotencyKey
required
string <uuid>

IdempotencyKey is for idempotent retry on POST.

maxFee
string

MaxFee is the max fee of EVM according to EIP-1559. This is obtained from estimate transaction fee API.

priorityFee
string

PriorityFee is the priority fee of EVM according to EIP-1559. This is obtained from estimate transaction fee API.

refId
string

RefID is the refId to associate the transaction. This ref Id will be a flexible field to support possible grouping that external endUser wants to maintain.

walletId
required
string <uuid>

WalletID is the source wallet ID of the transaction.

Responses

https://api-sandbox.circle.com/v1/w3s/developer/transactions/contractExecution

Request samples

  • Payload
Content type
application/json
{
  • "abiFunctionSignature": "burn(uint256)",
  • "abiParameters": [
    • "[\"100\"]"
    ],
  • "autoAccelerate": false,
  • "contractAddress": "string",
  • "feeLevel": "LOW",
  • "gasLimit": "21000",
  • "gasPrice": "string",
  • "idempotencyKey": "a0eebc99-9c0b-4ef8-bb6d-6bb9bd380a11",
  • "maxFee": "string",
  • "priorityFee": "string",
  • "refId": "string",
  • "walletId": "a0eebc99-9c0b-4ef8-bb6d-6bb9bd380a11"
}

Response samples

  • 200
  • default

{
  • "id": "string",
  • "state": "INITIATED"
}
